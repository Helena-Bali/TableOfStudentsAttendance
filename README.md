# Getting Started 

### `git clone git@github.com:Helena-Bali/TableOfStudentsAttendance.git`

In the root project directory run:

### `git clone git@github.com:Helena-Bali/client.git`

### `npm install`

### `cd client`

In the client directory run:

### `npm install`


In the root project directory run:

### `npm run dev`
